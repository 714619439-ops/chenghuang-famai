## 技术栈

- **纯静态 HTML5 + CSS3** — 无框架依赖
- **Google Fonts** — Noto Serif SC / Noto Sans SC / Ma Shan Zheng
- **GitHub Pages** — 自动部署托管
- **PWA** — 支持离线访问（基础 Manifest）
